# MyApkBuilder - Исходники для сборки APK

Этот проект был сгенерирован через **APK ZINE MAKER**. 
Так как облачный сервер не имеет Android SDK для прямой компиляции, мы подготовили для вас полностью готовый проект с настроенной авто-сборкой!

## Как получить готовый APK (Автоматически, без установки программ):

1. Распакуйте этот архив.
2. Создайте новый пустой репозиторий на [GitHub](https://github.com/new).
3. Загрузите все файлы из этой папки в ваш новый репозиторий.
4. Перейдите во вкладку **Actions** в вашем репозитории на GitHub.
5. Вы увидите запущенный процесс "Build Android APK".
6. Подождите 1-2 минуты. Когда процесс завершится зеленой галочкой, нажмите на него.
7. Внизу страницы в разделе **Artifacts** скачайте ваш готовый APK-файл!

## Как собрать локально (Для продвинутых):

Если у вас установлен Node.js, Java JDK и Android SDK:
```bash
npm install -g cordova
npm install
cordova platform add android
cordova build android
```
